vim回车换行

一月 6, 2021 发布在 windows, linux, vim

vim-problem

  1. 在windows下的文本文件的每一行结尾，都有一个回车('\n')和换行('\r')
  2. 在linux下的文本文件的每一行结尾，只有一个回车('\n');
  3. 而在linux下打开windows编辑过的文件，就会在行末尾显示^M;
  4. 注：^M在vim中为ctrl+M

解决：

vim中输入^M为：

i进入输入模式

ctrl+v ctrl+m

方法1：

vim 1.txt

:%s/\r//g

方法2：

^M输入方法为： ctrl+v ctrl+m

vim 1.txt

:%s /^M//g
